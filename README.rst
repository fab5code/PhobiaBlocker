PhobiaBlocker
=============

PhobiaBlocker is a web extension that blocks scary images of your phobia.
It's available in chrome and firefox.

Development
-----------

Create the extension folder *dist* with

.. code-block:: console

  $ npm run dev

or

.. code-block:: console

  $ npm run build

Then go to `chrome://extensions/ <chrome://extensions/>`__ and *load unpacked* the folder *dist*.
